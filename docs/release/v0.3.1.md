# CaracalDB v0.3.1 Release Notes

**Tag:** `v0.3.1`
**Date:** 2026-05-13
**Distribution:** `dist/caracaldb-0.3.1-py3-none-any.whl` + `dist/caracaldb-0.3.1.tar.gz`

## Highlights

- Promoted GNN sampling to official database APIs through
  `Database.sample_gnn_subgraph`, `Database.neighbor_loader`, and
  `Database.query_nodes`.
- Added PyG-style `(edge_index, n_id)` output where `edge_index` uses local
  node ids and `n_id` preserves global CaracalDB node ids, including isolated
  seed nodes.
- Extended `CsrReader.batch_neighbors` with documented fanout semantics:
  per-source uniform sampling, deterministic seeds, replacement sampling,
  `strategy="first"`, and `strategy="all"`.
- Updated the neighbor-loader PyG guide with the official CaracalDB-owned
  sampling workflow and semantics.

```powershell

```

## Compatibility

This is a patch release. Existing GraphRAG, vector search, traversal, and
storage APIs remain compatible with v0.3.0. The GNN sampling surface is newly
promoted and remains Python-reference backed.
