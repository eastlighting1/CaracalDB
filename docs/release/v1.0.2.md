# CaracalDB v1.0.2

v1.0.2 is a patch release for CSR/GNN sampling correctness and release-line
lint hygiene.

## Fixed

- Fixed `CsrReader.batch_neighbors(..., replace=True)` so replacement sampling
  preserves repeated selections instead of collapsing them to one edge.
- Kept the GNN neighbor-loader path ruff-clean for the v1.0 line.

## Install

```bash
uv add caracaldb==1.0.2
```

For direct pip-style installs through uv:

```bash
uv pip install caracaldb==1.0.2
```

Optional ML adapters remain available:

```bash
uv add "caracaldb[ml]==1.0.2"
```
