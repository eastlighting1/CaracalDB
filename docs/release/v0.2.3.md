# CaracalDB v0.2.3 Release Notes

**Tag:** `v0.2.3`
**Date:** 2026-05-02
**Distribution:** `dist/caracaldb-0.2.3-py3-none-any.whl` + `dist/caracaldb-0.2.3.tar.gz`

## Highlights

- Added a folder-rooted `.crcl` web viewer workflow. `caracal view` now defaults
  to `data/`, and the UI can discover and switch between `.crcl` files.
- Added direct path opening inside the viewer for files or bundles outside the
  discovered folder list.
- Updated the viewer's default inspection query to `MATCH (n) RETURN n LIMIT
  100`, with all-class inspection handled by the viewer API.
- Added a persisted light/dark theme toggle and refined the browser layout,
  table sizing, and dark-mode contrast.

## Build verification

```powershell
uv run --with hatch hatch build
# dist/caracaldb-0.2.3.tar.gz
# dist/caracaldb-0.2.3-py3-none-any.whl
```

## Smoke install

```powershell
uv pip install dist/caracaldb-0.2.3-py3-none-any.whl
python -c "import caracaldb as cdb; print(cdb.__version__)"  # 0.2.3
caracal --help
```

## Compatibility

This is a patch release. Existing 0.2.x resource ingest, generated diagnostics,
`connect`, `define_class`, `insert_nodes`, typed table ingest, and focused
`db.sql()` query APIs remain available.
