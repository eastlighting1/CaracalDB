# CaracalDB v0.1.3 Release Notes

**Tag:** `v0.1.3`
**Date:** 2026-04-29
**Distribution:** `dist/caracaldb-0.1.3-py3-none-any.whl` + `dist/caracaldb-0.1.3.tar.gz`

## Highlights

- `db.define_class()` now accepts `superclass_iris` for public ontology hierarchy registration.
- `db.sql()` can execute the focused class-closure predicate `alias.class SUBCLASSOF* <IRI>`.
- `SUBCLASSOF*` can be combined with ordinary `AND` filters in the MVP single-node query path.
- Ontology docs and the 30-Minute Tour now separate implemented class closure from broader experimental reasoning features.
