# CaracalDB v0.2.0 Release Notes

**Tag:** `v0.2.0`
**Date:** 2026-04-30
**Distribution:** `dist/caracaldb-0.2.0-py3-none-any.whl` + `dist/caracaldb-0.2.0.tar.gz`

## Highlights

- Added flexible resource ingest for Neo4j-style JSON objects, IRI resources, raw triples, typed node rows, and typed edge rows.
- Added `insert_triples`, `import_resource`, `import_resources`, `resource`, and `export_resource_turtle` to the public `Database` API.
- Introduced `caracaldb://resource/...` as the default display form for CaracalDB internal resource links.
- Kept IRIs optional for GNN/property-graph workflows while preserving explicit IRI metadata when provided.

## Build verification

uv run --with hatch hatch build

## Smoke install

```powershell
uv pip install dist/caracaldb-0.2.0-py3-none-any.whl
python -c "import caracaldb as cdb; print(cdb.__version__)"  # 0.2.0
caracal --help
```

## Compatibility

This is a minor release. Existing 0.1.x `connect`, `define_class`, `insert_nodes`, typed table ingest, and focused `db.sql()` query APIs remain available.
