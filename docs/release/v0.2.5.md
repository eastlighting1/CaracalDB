# CaracalDB v0.2.5 Release Notes

**Tag:** `v0.2.5`
**Date:** 2026-05-06
**Distribution:** `dist/caracaldb-0.2.5-py3-none-any.whl` + `dist/caracaldb-0.2.5.tar.gz`

## Highlights

- Added direct `pyarrow.Table` ingestion for node and edge table APIs.
  `insert_nodes`, `insert_node_table`, and `insert_edge_table` now accept Arrow
  tables alongside mapping rows.
- Added explicit Arrow-first helpers: `insert_node_table_arrow` and
  `insert_edge_table_arrow`.
- Added `node_table` and `edge_table` APIs so stored node and edge batches can
  be read back as Arrow tables.
- Added simple, weighted, and complex example `.crcl` bundles for viewer and
  documentation workflows.
- Reorganized API reference pages and refreshed Quickstart, tour, guide, and
  concept examples with executable snippets and package-relative paths.
- Improved documentation diagrams, theme palette behavior, and quickstart code
  checking.

## Compatibility

This is a patch release. Existing 0.2.x resource ingest, snapshot reads,
`.crcl` viewer, generated diagnostics, `connect`, `define_class`, row-based
table ingest, and focused `db.sql()` query APIs remain available.
