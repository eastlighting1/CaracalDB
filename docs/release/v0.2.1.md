# CaracalDB v0.2.1 Release Notes

**Tag:** `v0.2.1`
**Date:** 2026-04-30
**Distribution:** `dist/caracaldb-0.2.1-py3-none-any.whl` + `dist/caracaldb-0.2.1.tar.gz`

## Highlights

- Polished public documentation after an external writing-quality review, including Quickstart-adjacent start pages, concept pages, ADRs, tutorials, and the developer overview.
- Registered documented CDB diagnostics in the runtime diagnostics table so generated error docs include the same CDB codes as the public per-code pages.
- Kept generated docs checks clean by moving error-index updates into the generator source path instead of hand-editing the generated file.

## Build verification

```powershell
uv run --with hatch hatch build
# dist/caracaldb-0.2.1.tar.gz
# dist/caracaldb-0.2.1-py3-none-any.whl
```

## Smoke install

```powershell
uv pip install dist/caracaldb-0.2.1-py3-none-any.whl
python -c "import caracaldb as cdb; print(cdb.__version__)"  # 0.2.1
caracal --help
```

## Compatibility

This is a patch release. Existing 0.2.0 resource ingest, `connect`, `define_class`, `insert_nodes`, typed table ingest, and focused `db.sql()` query APIs remain available.
