# Errors

Error documentation explains what a user sees, why it happens, how to fix it, and which related concepts matter.

## Code Families

- `TF-*`: Tuft language diagnostics.
- `CDB-*`: engine, storage, transaction, ontology, and I/O errors.

The error index is generated from the runtime diagnostic table so the site stays aligned with code.

Start with the generated [Error Index](index.md), then use per-code pages once a diagnostic needs fuller repair guidance.
