# CaracalDB v0.1.0 — Release Notes

**Tag:** `v0.1.0`
**Date:** 2026-04-27
**Distribution:** `dist/caracaldb-0.1.0-py3-none-any.whl` + `dist/caracaldb-0.1.0.tar.gz`

## Highlights

- Embedded, Arrow-native graph database hits the **M5 gate**.
- 270+ pytest checks, 4 micro-benchmarks gated against `bench/results/baseline.json`.
- `caracal` CLI (`init / run / explain / bench`) plus three executable tutorial notebooks (`examples/*.ipynb`).
- Full milestone history pinned in `docs/milestones/M0…M5-gate.md`.

```powershell
caracal init demo.crcl
```

## SemVer commitments

- The wire format (`MAGIC=CRCL\0\0\0\1`, MANIFEST, CSR layout) is now
  considered stable across the 0.1.x line.
- Public Python API surfaces (`cdb.connect`, `Connection.sql`,
  `NeighborLoader`) preserve their signatures across patch releases.
- Error codes (`TF-`, `CDB-`) are stable identifiers; messages may evolve.

## Carry-overs to v0.2.0

User docs (CDB-089), error catalogue (CDB-090), API reference (CDB-091),
and the `caracal repl` interactive shell.
