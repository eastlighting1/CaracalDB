# CaracalDB v0.1.1 Release Notes

**Tag:** `v0.1.1`
**Date:** 2026-04-29
**Distribution:** `dist/caracaldb-0.1.1-py3-none-any.whl` + `dist/caracaldb-0.1.1.tar.gz`

## Highlights

- MkDocs documentation can now be deployed through GitHub Pages.
- Public documentation code fences render correctly, including Python, Tuft, Mermaid, Bash, and text blocks.
- Documentation validation now fails on malformed closing fences.
- Public milestone documentation no longer depends on internal milestone gate reports.
- Error index hints are generated from `caracaldb.lang.diagnostics.ERROR_TABLE`.

## Build verification

```powershell
uv run --with hatch hatch build
# dist/caracaldb-0.1.1.tar.gz
# dist/caracaldb-0.1.1-py3-none-any.whl
```

## Smoke install

```powershell
uv pip install dist/caracaldb-0.1.1-py3-none-any.whl
python -c "import caracaldb as cdb; print(cdb.__version__)"  # 0.1.1
caracal --help
```

## Compatibility

This is a patch release. The Python public API and documented 0.1.x storage-format commitments remain unchanged.
