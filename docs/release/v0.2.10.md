# CaracalDB v0.2.10 Release Notes

**Tag:** `v0.2.10`
**Date:** 2026-05-08
**Distribution:** `dist/caracaldb-0.2.10-py3-none-any.whl` + `dist/caracaldb-0.2.10.tar.gz`

## Highlights

- Added `entity_degree_prior`, `entity_evidence_prior`, and `evidence_direction` scoring parameters to `graphrag_search` to influence entity candidate ranking and bounded evidence expansion.
- Added `entity_linking_ms` profiling metric to `graph_ecosystem` benchmark reporting.
- Significantly improved graph traversal performance by caching adjacency index lookups (including filtered and ordered lookups) and indexing graph priors.
- Optimized `evidence_search` queue traversal with O(1) shifts and early stopping based on target node type.
