# CaracalDB v0.1.4 Release Notes

**Tag:** `v0.1.4`
**Date:** 2026-04-29
**Distribution:** `dist/caracaldb-0.1.4-py3-none-any.whl` + `dist/caracaldb-0.1.4.tar.gz`

## Highlights

- `db.define_class(..., superclass_iris=...)` now updates existing class definitions by merging superclass metadata.
- Existing `.crcl` databases can be reopened and upgraded with ontology hierarchy metadata without being recreated.
- Added regression coverage for the reopen-and-upgrade path used by ontology examples.

## Build verification

```powershell
uv run --with hatch hatch build
# dist/caracaldb-0.1.4.tar.gz
# dist/caracaldb-0.1.4-py3-none-any.whl
```

## Smoke install

```powershell
uv pip install dist/caracaldb-0.1.4-py3-none-any.whl
python -c "import caracaldb as cdb; print(cdb.__version__)"  # 0.1.4
caracal --help
```

## Compatibility

This is a patch release. Existing 0.1.x query, packed database, and storage APIs remain available.
